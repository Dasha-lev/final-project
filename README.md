# Фінальний проєкт

Цей проєкт містить повну інфраструктуру як код (IaC) для розгортання масштабованого Django-застосунку в AWS з використанням Terraform, Kubernetes (EKS), CI/CD та системи моніторингу.

---

## 🏗️ Архітектура та компоненти
- **VPC:** Безпечна мережа з поділом на публічні та приватні підмережі, NAT Gateway та інтернет-шлюзом.
- **EKS:** Кластер Kubernetes для оркестрації контейнерів додатка та сервісів автоматизації.
- **RDS / Aurora:** Гнучкий модуль бази даних з підтримкою умовної логіки (`use_aurora`).
- **ECR:** Приватний реєстр для зберігання Docker-іміджів.
- **Jenkins & Argo CD:** GitOps-інструменти для автоматичної збірки та деплою.
- **Prometheus & Grafana:** Комплексний моніторинг стану кластера та збір метрик.
